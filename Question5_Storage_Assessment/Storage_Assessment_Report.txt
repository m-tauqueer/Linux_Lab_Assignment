Storage Health Observations:
The root filesystem is at 44% usage with 120G free, which is healthy.
Inode usage at 8% is well within safe limits. No filesystem is above 80%.
zram provides compressed swap for extra memory flexibility.

Recommendations for Improving Storage Utilization:
1. Periodically check large directories with: du -sh *
2. Clear the pacman package cache: sudo pacman -Sc
3. Vacuum old system logs: sudo journalctl --vacuum-time=2weeks
4. Move large media to the unused 249.2G partition or external storage.
5. Set an alert when root usage crosses 80%.
